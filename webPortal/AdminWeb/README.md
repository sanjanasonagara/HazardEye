# HazardEye Admin Dashboard

Frontend-only Admin Dashboard for the HazardEye industrial safety platform.

## Features
- **Dashboard**: High-level KPIs and recent activity.
- **Analytics**: Visualization of incident trends, severity, and areas using Recharts.
- **CMS**: Content management for safety guidelines and SOPs.
- **Reports**: Summary views and mocked download options.
- **CS System**: Case management and support ticket monitoring.

## Tech Stack
- React (Vite)
- Tailwind CSS
- Recharts
- Lucide React Icons
- React Router DOM

## Setup & Run

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Folder Structure
- `src/components`: Reusable UI components.
- `src/pages`: Page views (Dashboard, Analytics, etc.).
- `src/data`: Mock data.
- `src/layouts`: Layout wrappers.
